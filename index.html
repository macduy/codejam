What is it?

codejam is a small, easy-to-use, Python framework for writing solutions to codejam competition problems. It takes the hassle out of writing code, so that you can concentrate on what is important: solving problems.

If you use Python to solve codejam problems, this might be the right tool for you!

The following is the complete solution to Qualification Round 2011, Problem B

INPUT = {
    'total': ('int', 'constant'),
    'candies': ('int', 'linearray')
}

INPUT_ORDER = ['total', 'candies']

def main(total, candies):
    if reduce(lambda a,b: a^b, candies) != 0:
        return "NO"
    else:
        candies.sort()
        candies.reverse()
        candies.pop()
        return sum(candies)
